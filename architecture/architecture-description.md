# Architecture

## Components

1. User
2. Compliance Guardian Copilot
3. Work IQ
4. Knowledge Sources
5. Analysis Engine
6. Microsoft 365 Services

## Workflow

User submits a contract or compliance query.

↓

Copilot Studio receives the request.

↓

Work IQ retrieves relevant enterprise context.

↓

Knowledge sources provide policies and reference documents.

↓

AI analyzes the content.

↓

Risk assessment and compliance gap detection are performed.

↓

Results are presented as:

- Executive Summary
- Risk Score
- Compliance Findings
- Recommended Actions

↓

Optional workflow actions:
- Teams notification
- Outlook email
- Compliance review tasks
